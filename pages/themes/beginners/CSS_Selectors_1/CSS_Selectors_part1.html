Какво са CSS селекторите?

Шаблони, които определят над кои HTML елементи ще се приложи зададеният стил в декларациите.
Mогат да бъдат съвсем прости, като '*' или 'p' или доста по-сложни като: 
a[href^="https://"]:hover

Списък със всички дефинирани в стандарта (CSS1 – CSS3) селектори: css_selectors

В момента (2016) се разработва спецификацията
CSS Selectors Level 4

Селектори за елементи

Element (type) selectors


            /* Всички параграфи в документа ще бъдат с червен шрифт */
            p {
                color: red;
            }

Действие: селектира всички HTML елементи от дадения тип.

Пример: "Цитати на известни личности"

Универсален селектор (Universal selector)


            * {
                marging: 0;
                padding: 0
            }

Действие: селектира всички HTML елементи в документа.

Селектори за class и id

Селектор за class

Записва се с точка (".") последвана от стойност на class.


            /* Селектира всички HTML елементи, които са дефинирали class="blue" */
            .blue{
                color: blue;
            }

Може да бъде специфициран допълнително, като пред него непосдредствено се запише тип на елемент, 


            /* Селектира всички HTML елементи, от тип 'p', които са дефинирали class="red" */
            p.red{
                color: red;
            }

Селектор за id

Записва се с диез (хаш таг) - "#", последван от стойност на атрибута id.

Тъй като с дадено id можем да обозначим само един единствен елемент от цялто HTML дърво, то е безсмислен записът с тип на елемент отрпед.

Например: p#vip{...} няма смисъл и не се препоръчва да се използва.

Пример


        /* Селектира всички HTML елементи, които са дефинирали class="c1" */
        .c1{
            color: purple;
        }
        /* Селектира всички HTML елементи, от тип 'h1', които са дефинирали class="c1" */
        h1.c1{
            font-style: italic;
        }

        /* Селектор по id */
        #vip{
            color: #aa3333;
            font-size: 2em;
            text-shadow:
                2px 2px #a000a0,
                2px 2px #c000c0;
        }

CSS Комбинатори (CSS Combinators)

Това са селектори отчитащи позицията (контекста) на елементите в HTML дървото.

Видове HTML възли

Родител (Parent)
Дете (Child)
Наследници (Descendants)
Братя (Siblings)

  • Пример

Поради съображения за сигурност не можем да селектираме елементи и в ляво по дървото. Поради това няма селектор за предшественик.

Селекторите, чрез които можем да избираме елементи в HTML дървото са:

  • селектор за наследници (descendant selector): 'space'
  • селектор за деца (child selector): '>'
  • селектор за брат - без значение дали следва непосредствено брат си (general sibling selector): '~'
  • селектор за брат, който е разположен непосредствено след брат си (adjacent sibling selector): '+'
